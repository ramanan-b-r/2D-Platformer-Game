# 2D-Platformer-Game
This is a classic 2D platformer-style game with 3 platforms. The game has two modes: local PvP multiplayer and local single-player.

# CONTROLS 

Player 1 Controls:

Jump: W\
Move Left: A\
Move Right: D\
Shoot: Left Shift

Player 2 Controls:

Jump: Up Arrow\
Move Left: Left Arrow\
Move Right: Right Arrow\
Shoot: Right Shift

General Controls:
Escape: Quit the game
